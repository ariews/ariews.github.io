VLC:2

Arie W. Subagja
2005-08-06
Tutorial, Warnet

Setelah kemaren install VLC di Xandros, skr pas gw mo install di “Hoary Hedgehog” [kompi na si pocong], gw kebingungan.

loh.. koq?

Iya, soalna [selain pake Gnome, juga] tampilannya beda pisan!! Ternyata ada di “Synaptic Package Manager”, gw langsung pilih Multimedia, tapi ternyata gak ada. Pilih Search, n masukan keyword “wmv” [kayak kemaren pas di xandros], hasilnya NOL BESAR!

Gak sempet baca Support, yang gw baca adalah file /etc/apt/sources.list, ada tulisan archives, ah coba2 aja dulu, sapa tau ada. Soalna kan rada lama neh vlc nya :)

## Uncomment the following two lines to add software from the 'universe'
## repository.
## N.B. software from this repository is ENTIRELY UNSUPPORTED by the Ubuntu
## team, and may not be under a free licence. Please satisfy yourself as to
## your rights to use the software. Also, please note that software in
## universe WILL NOT receive any review or updates from the Ubuntu security
## team.
#deb http://us.archive.ubuntu.com/ubuntu hoary universe
#deb-src http://us.archive.ubuntu.com/ubuntu hoary universe

Uncomment 2 baris terakhir, lalu update.

Gw test tik: sudo apt-get -s install vlc eh ternyata kluar.

Ya udah sekalian install
