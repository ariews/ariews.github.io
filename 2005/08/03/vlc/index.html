VLC

Arie W. Subagja
2005-08-03
Tutorial, Warnet

Gw di kasih pelem dot-wmv sama temen, katanya bagus. Sialnya gw gak bisa buka di xandros ini.

Gw test pake VideoPlayer… kagak bisa, pake Xine… kagak bisa, pake Real Audio kagak bisa juga, tus gw install Media Playernya windows (pake CrossOver) eh… masih kagak bisa juga. Nasib nasib…

Yang pake windows kali ketawa2 liat keadaan gw, ugh… kejam. Kayak si BijiKodok, dia ngakak2 pas tau disini gak bisa buka file dot-wmv.

Gak putus asa, gw buka XandrosNetwork (XN), search wmv, ketemu vlc.

Ok… gw test.
